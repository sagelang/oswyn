# Oswyn — Your Sage Companion

A browser-based AI assistant that knows everything about the [Sage programming language](https://github.com/sagelang/sage).

**[Try Oswyn →](https://sagelang.github.io/oswyn/)**

## Features

- Ask questions about Sage syntax, agents, LLM integration, tools, testing, and more
- Get working code examples tailored to your question
- Conversation memory across sessions (stored in localStorage)
- Supports OpenAI, Anthropic, and any OpenAI-compatible API
- No backend — everything runs in your browser
- Your API key never leaves your browser

## Setup

1. Open [sagelang.github.io/oswyn](https://sagelang.github.io/oswyn/)
2. Click the settings icon and enter your API key
3. Start chatting!

## Privacy

Oswyn runs entirely in your browser. Your API key and conversation history are stored in localStorage and are never sent to any server other than your chosen LLM provider.
